Student Management System

A Student Management System built using Python and Django with MongoDB as the database to manage student records and academic information efficiently. This web-based application helps educational institutions store, update, and retrieve student data in a scalable and flexible manner.

Features

* User Authentication
* Add, Update, and Delete Student Records
* Manage Classes, Courses, and Departments
* Track Student Enrollment Details
* Clean and user-friendly web interface

Tech Stack

Backend: Python, Django
Database: MongoDB
ORM: PyMongo 
Frontend: HTML, CSS, Bootstrap
